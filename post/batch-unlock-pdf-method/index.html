批次解鎖受保護的PDF檔

July 13, 2021

目的

解除大量鎖定列印與編輯的PDF檔案

必要軟體

ghostcript

  1. 下載連結 https://www.ghostscript.com/download/gsdnld.html
  2. 選擇Ghostscript 9.54.0 for Windows (64 bit) Ghostscript AGPL Release
  3. 標準安裝完(基本上都是下一步點一點)
  4. 標準預設目錄在C:\Program Files\gs\gs9.54.0

新增系統路徑

  1. win + S > 搜尋環境變數
  2. 選擇編輯系統環境變數
  3. 新增使用者變數 變數名稱 path 變數值 C:\Program Files\gs\gs9.54.0\bin
  4. 確定

命令提示字元操作說明

  1. 在C槽先建立一個工作目錄(例如: C:\pdf)
  2. win + R > 執行
  3. 輸入cmd > 確定
  4. 開啟命令提示字元
  5. cd C:\pdf (調整至工作目錄 )
  6. 畫面顯示C:\pdf>
  • 在cmd中要貼上可以用滑鼠右鍵貼上

單一解鎖方式

  1. 在工作目錄放入想解鎖的pdf
  2. 在cmd中輸入 (C:\pdf>)
gswin64c -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sOutputFile="1-unlocked.pdf" "1.pdf"

3. 按下enter後完成

批次解鎖方式

  1. 在工作目錄放入想解鎖的pdf們
  2. 建立conv資料夾用於存放解鎖後的pdf
  3. 建立一個conv.txt
  4. 建立一個conv.bat (注意附檔名要是.bat)
  5. 可以用檢視>顯示/隱藏>勾選副檔名，以免建立成conv.bat.txt之類的情況
  6. 在cmd中輸入 (C:\pdf>)
dir *.pdf /b >conv.txt

  1. 此時conv.txt有檔名列表

  2. 在cmd中輸入 (C:\pdf>)

echo FOR /F "eol=; usebackq delims==" %%i in (conv.txt) do gswin64c -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sOutputFile="conv/%%i" "%%i" > conv.bat

  1. 在cmd中輸入 (C:\pdf>)
conv.bat

10. 所有輸出檔案即在子資料夾conv裡面
